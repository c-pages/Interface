void app::Application::actualiser ( float  deltaT)

Actualiser les éléments.

Actualiser les différents éléments du ou des écrans actifs.

Paramètres
deltaTUn float qui indique le delta du temps écoulé depuis la dernière actualisation. ca permet quand on actualise de ponderer les mouvement en fonction du temps et ainsi avoir une indépendance entre animation et frame rate.
