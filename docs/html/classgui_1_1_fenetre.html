Gadget permettant de rassembler des Gadgets au sein d'une Fenetre.

// //
// HORIZ_... //
// //

// | | | | | //
// 0 --- |----|-----------------------|----| --- //
// |Red.| Redimension |Red.| m_bordure //
// 1 --- |----|-----------------|-----|----| --- //
// | | | | | m_lblTitre->getSize().y //
// | | Titre/drag |Ferm.| | --- //
// | | | | | m_bordure //
// 2 --- | 0.0----------------|-----| | --- //
// |Red.| |Red.| //
// | | | | //
// | | Contenu | | taille.y //
// | | ( taille ) | | //
// | | | | //
// 3 --- |----|-----------------------|----| --- //
// |Red.| Redimemsion |Red.| m_bordure //
// Total --- |----|-----------------------|----| --- //
// //
// | | | | //
// m_bordure taille.x m_bordure //
